# 20201112 作業：Discord 登入系統
請撰寫一個程式，在輸入正確的帳號密碼時顯示隨機歡迎訊息。
如果帳密錯誤，請讓使用者可以重新輸入。

程式中請使用兩組陣列，分別為：

1. users：帳號、密碼、姓名三個欄位。
2. messages：接在名稱後面的歡迎詞。

可以自行設計寫死在程式碼中，名稱可以固定於歡迎詞前或後。

![](https://i.imgur.com/TWlCKK4.jpg)


## 輸入說明
輸入預先設定於程式碼中的帳號與密碼組。

## 輸出說明
如果帳密正確，顯示名稱與歡迎詞。

如果帳密錯誤，請讓使用者可以重新輸入。

## 範例

### 帳密正確範例

帳號：*goodjack*

密碼：*Goodjack-Is-Handsome!*

小克 剛剛滑入了伺服器中。

### 帳密錯誤範例

帳號：*goodjack*

密碼：*Goodjack-Is-Not-Handsome:(*

帳號密碼錯誤，請重新輸入。
